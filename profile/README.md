# 🧬 OpenCure Labs

**Autonomous AI agents running real science — so every researcher can fight cancer like a funded lab.**

---

## 🎯 The Mission

A developer named Paul Conyngham used open-source tools to design a personalized cancer vaccine that saved his dog Rosie. He proved one person with the right pipeline can do what takes an institution months.

We're building the infrastructure so **anyone, anywhere** can run that same pipeline — and push it further. Personalized medicine shouldn't require a grant and a team of 50.

## ⚙️ How It Works

Specialist AI agents coordinate, critique, and iterate — autonomously. No human in the loop unless you want one.

1. **Coordinator** dispatches tasks to domain agents (cancer, rare disease, drug response)
2. **Science agents** run pipelines — VCF parsing, epitope prediction, docking simulations
3. **Critics** (Claude Opus 4.6 + Grok) review every result for scientific rigor
4. **Results stream live** to Discord and get published as reproducible reports
